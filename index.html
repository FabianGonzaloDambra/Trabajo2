Descripción del proyecto

Este proyecto nace dentro de la cátedra de Prácticas Profesionalizantes 2 con el fin de diseñar un software para la empresa "Ernesto Vazquez Automores".

En el mismo se detallan todas las fases del ciclo de desarrollo del software así como los programas utilizados y los lenguajes de programación requeridos. Toda la información recabada se vuelca en una página web funcional. Para ella se utilizan HTML para el maquetado, CSS para los estilos y javascript para las interacciones y animaciones.
