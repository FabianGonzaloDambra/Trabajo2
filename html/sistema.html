Análisis de las necesidades del sistema

  • El software debe ser capaz de contar con una base de datos de todos los vehículos clasificados por marca y modelo e incluir en ella además todos los precios directos e indirectos asociados a cada uno para llevar la contabilidad así como de los repuestos y reparaciones.
  • El software debe incluir en la base de datos, los datos de los clientes, permitiendo las altas y bajas además de facilitar el reconocimiento y asociar al cliente con sus productos.
  • El software debe llevar una base de datos de todos los empleados internos y externos a la empresa, y permitir su correcta identificación.
  • El software debe ser accesible solo por el sector correspondiente en cada caso y a su vez manejar un panel de información general.
